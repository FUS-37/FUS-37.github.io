第0章 序

大巴车缓缓开远，很快就在林间的公路上消失不见，没入这里——层层叠叠的山林之地。

在这么孤僻的地方，平均半个月才会有一班大巴车。尤其是最后的这段路，按说车上应该是没什么人了，换作平时，乘客们基本在上一站之前就全下了车。

在最后的这段林间公路，车上只剩下我和司机两个人。在上一站最后一个老奶奶下车后，车上的氛围开始变得沉默起来。他会好奇我是来做什么的吗？一看就是外地的面孔，毕竟那个司机好像和所有乘客都早已熟悉。更何况我身上还带着一个大号的卡其布背包和一只MYIO行李箱。一个20出头的女生只身一人带着这么多行李，一路上没少被人问起。

安静。

我手里盘着一串老桃木珠子，撇头看向窗外。夏季的山里空荡又惬意，阴凉的风穿过，伴着不知来自何处的鸟鸣。山里弥漫着雾气，阳光透进雾里，照出今晨刚下过的雨水的痕迹。如此这般，我来到了这片山林。

从车站往山的方向走去，不多时就能看到石阶。沿着石阶向上，很快就能望见那座寺门的遗迹。穿过寺门是一块平地。平地上建着一排房屋，围成一圈，中间是一个不大不小的庭院。或许有人想知道这山叫什么名字，听说寺叫圆寺，那这山姑且叫圆山吧。仿佛名字能把它同其他不知名的小山来区分开。

名字又有何必须呢？

我只是一个来寻找灵感的音乐人，碰巧遇上了一片无人的山林。姑且打算是隐居起来，不过还是走一步看一步的好。虽说今年音乐学院开学要到10月，但其实我并没有把握呆到那个时候。毕竟自从出生以后，这是我第一次远离人群。

我从小就是很奇怪的一个人，在团体中总是很容易就把自己摆在一个旁观的位置。但即便如此，我也没有试着逃离过群体。进入高中以后，我已经能在群体生活中游刃有余地寻找空隙。当时中午都喜欢呆在实验室旁的楼梯转角，这大概就是我最接近一个人的时刻了。从那里一直望去，是学校的操场，目光再向远处是湖畔、亭台与熹微着的林光。在那里我度过了无数一个人的中午，一段偷摸着听音乐的时光。

到了现在也是一样，我好像一直在找适合旁观风景的地方。话说，从远处旁观到的风景会让你联想到什么？是认知到只有一人独自清醒而感到自得的冲动，偷窥着自己所觊觎的情感，但这一过程中又确保自己在别人的记忆中不会存在。所谓旁观，说是最强有力的不在场证明也不过分了。

人的记忆往往趋向于自己所已然相信的。哪怕存在不复，只要留有回忆，就不会突然化为虚无，不，甚至有些时候正因种种缺憾，更促着回忆日益强烈而肆意浓郁。

行李很多，尤其是箱子里的那套录音器材、声卡和一把midi键盘。出门旅行并不高兴再带上什么乐器呢，太占地方。我就只是打算录一些山林里的声音回去而已。路上看到山脚的小镇也很有意思，沿着两条江流蜿蜒的方向而筑成的村落，不知从山上看又会是什么样的景色呢。不过在一切开始之前，我得先把今天晚上落脚的地方收拾出来。

走进房间，寺里比我想象得干净。庭院一尘不染，像是一直都有人打扫的痕迹。踏入山门是几间屋舍，挡住通往后山的小路。

据说我可以随便挑一个空的屋子住下，有后山的溪泉水足以满足日常用度，甚至在溪流上游架设了发电机用以供电。说来倒是梦幻，都是如此偏僻的地方了，网络信号却并不差。

在我的隔壁屋子似乎有人居住，不过那人应该没有什么行李，书架上书倒是挺多，除此之外还有一本《金阁寺》就那么明晃晃地摊在桌子上。事实上我的行李也没多到哪去。除了那些设备，我的个人衣物也没有带几件。毕竟现在是夏天，起码不会说容易着凉。

『不过目前还是先收拾一下屋子吧。』

我这样想着，刚放好东西，顺手看了一眼手机，有一封未读邮件提示：

『......因为一些众所周知的原因，......您先前订阅的音乐制作套件服务将不再继续更新并且永久关闭......衷心祝愿在以后的日子里您能过得愉快。』
